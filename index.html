Welcome to RAGE Github Page.

Welcome to R.A.G.E. - Ruby Awesome Game Engine, written in C++ using Ruby language and Allegro Game Library. You can make all kinds of 2D games using RAGE engine.

You use Ruby, an Interpreter language to handle all game logic, most of the heavy lifting is done by the engine itself so you can concentrate more on making awesome games. Enjoy!
